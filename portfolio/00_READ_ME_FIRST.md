## 🎨 Customize in Minutes

### Update Your Information

Edit `src/data/content.ts`:

```typescript
export const personalData = {
  name: 'Your Name',
  email: 'your.email@gmail.com',
  github: 'your-github-url',
  // ... update all fields
};
```

### Change Colors

Edit `tailwind.config.ts`:

```typescript
colors: {
  primary: '#0099ff',      // Your primary color
  secondary: '#7c3aed',    // Your secondary color
  accent: '#06b6d4',       // Your accent color
}
```

### Update Experience, Projects, Skills

Everything is in `src/data/content.ts` - just edit and refresh!
